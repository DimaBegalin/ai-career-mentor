# AI Career Mentor

Квест-профориентация для школьников: 13 вопросов → специальности по ответам →
живой ИИ-поиск вузов под баллы IELTS/SAT → чек-лист в WhatsApp через WABA-бота.

## Структура проекта

```
AI-Career/
├── server.js              сервер: статика + прокси к ИИ + приём лидов + ЛОГИ
├── package.json           команды запуска
├── .env.example           шаблон настроек (скопировать в .env)
├── logs/app.log           файл логов (создаётся при запуске)
├── public/                фронтенд
│   ├── index.html         разметка
│   ├── css/style.css      дизайн-система (тёмная EdTech-тема)
│   └── js/
│       ├── config.js      эндпоинты и модель
│       ├── logger.js      логгер: консоль + сервер + панель ?debug=1
│       ├── data.js        вопросы, специальности, фолбэк-вузы
│       ├── logic.js       состояние, скоринг, фолбэк-результат
│       ├── ai.js          промпт + вызов ИИ с веб-поиском
│       └── app.js         действия, рендер экранов, лид-форма
└── n8n/                   WhatsApp-бот
    ├── whatsapp-checklist-bot.json   воркфлоу для импорта
    └── README-whatsapp-bot.md        настройка WABA + Meta
```

## Быстрый старт

```bash
cd ~/Desktop/AI-Career
cp .env.example .env        # впишите свой ANTHROPIC_API_KEY
node server.js              # нужен Node.js 18+
```

Откройте http://localhost:3000 — сайт работает.
Без ключа в .env всё тоже работает: ИИ-поиск переключится на встроенную
фолбэк-подборку вузов (в логах будет видно, какой путь сработал).

## Где смотреть логи

1. **Терминал** — живые логи сервера и клиента ([CLIENT]) сразу при `node server.js`:
   ИИ-запросы с длительностью, каждый ответ ученика, новые лиды (номер маскируется).
2. **Файл** — `logs/app.log`, вторая вкладка терминала: `npm run logs`
   (или `tail -f logs/app.log`).
3. **Браузер** — откройте `http://localhost:3000/?debug=1` — внизу появится
   панель логов прямо поверх сайта. Удобно смотреть на телефоне.
4. **n8n → Executions** — каждое срабатывание WhatsApp-бота.

## Деплой

Любой Node-хостинг (Railway, Render, VPS): залить папку, задать переменные
окружения из .env, команда запуска `node server.js`. Фронтенд и API живут
на одном домене, CORS не нужен.
